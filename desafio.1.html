História dos Computadores

Desde os primeiros computadores com válvulas e cartões perfurados até os dispositivos portáteis atuais, a história da computação é marcada por uma evolução impressionante.

Esse progresso permitiu que a web surgisse e se tornasse essencial para a vida moderna.

“O computador moderno é resultado de décadas de desenvolvimento tecnológico e teórico.”

O que são Clients?

Um client é o usuário final que acessa um sistema através de um navegador, aplicativo ou outro meio. Ele solicita informações a um servidor e recebe os dados em troca.

Exemplo: seu navegador acessando o Google é um cliente.

O que são Servers?

Um server é um computador que responde às requisições de múltiplos clients. Ele pode entregar páginas HTML, imagens, vídeos ou processar dados.

Imagine-o como um garçom digital que entrega o que você pediu na web.

Estrutura básica do HTML

Todo arquivo HTML começa com a declaração <!DOCTYPE html>, indicando o tipo de documento. Em seguida, usamos:

  1. <html>: raiz da página
  2. <head>: contém metadados e título
  3. <body>: conteúdo visível ao usuário

Veja um exemplo de formatação:

H2O é a fórmula da água, e x2 representa um número ao quadrado.

Listas Ordenadas e Não Ordenadas

As listas ordenadas usam números:

  1. Item 1
  2. Item 2
  3. Item 3

As listas não ordenadas usam marcadores:

  • Elemento A
  • Elemento B
  • Elemento C

Tags Especiais e Semânticas

Algumas tags são menos usadas, mas podem ser úteis:

<font> (obsoleta)

Este texto está azul usando a tag <font> (não recomendada em HTML5)

<del>

Esta informação estava errada: HTML é uma linguagem de programação. Na verdade, é de marcação.

<abbr>

A sigla HTML é o alicerce da web.
